# 上汽乘用车大数据交互平台
## 技术栈
```
vue2.6.10
vuex3.1.1
vue-router3.0.6
element-ui2.9.2
axios0.19.0
echarts4.3.0
```

## 框架功能
```
1.支持自定义菜单展示方式： => 顶部菜单、侧边栏菜单、tab菜单
2.支持自定义设置是否全屏展示
3.支持自定义设置是否显示收缩展示侧边栏按钮
4.面包屑导航(前进刷新，后退缓存页面不刷新)
5.登录注销
6.鉴权
7.支持响应式设计布局
8.动态更新记录页面缓存与否

```
